# API Test Automation Project

This project contains automated UI tests written in Java using Selenium and Maven.

## 🔧 Technologies
- Java
- Selenium WebDriver
- Maven
- Page Object Model (POM)

## 📁 Project Structure
- `src/test/java` — test classes
- `core` — base classes and setup
- `helpers` — utility classes

## ▶️ How to Run Tests
1. Open project in IntelliJ IDEA
2. Install dependencies (Maven)
3. Run test classes

## 💡 About
This project was created as part of QA Automation learning and practice.
